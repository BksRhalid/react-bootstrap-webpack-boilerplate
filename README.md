# React Boilerplate

## Features
- Webpack building system
- Webpack Dev Server with Hot reload
- Bootstrap 3

## Webpack Loaders
- URL
- File
- Styles: LESS, CSS
- Imports (shimming modules)
- Babel (ex. 6to5) Loader for JSX and ES6
- React Hot Loader

## Installing
`npm install`

## Running
run `npm start` then check in your browser at http://localhost:8080
